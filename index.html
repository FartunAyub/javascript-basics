JavaScript Basics Assignment

Part 1: Data Types

1. Create variables of different data types: string, number, boolean, null, undefined, object, and symbol.

2. Print these variables to the console using console.log().

Part 2: Arrays

1. Create an array with various elements (numbers, strings, etc.).

2. Access and modify array elements.

Part 3: Functions

1. Create a function that performs a specific task.

2. Call the function and use its output.
